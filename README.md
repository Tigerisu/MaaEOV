## 功能列表

- 启动游戏
- 每周-阈限禁区
- 清体力
- 危机行动调令
- 事务所（解读仪式）
- 市区巡逻
- 领取任务奖励
- 领取应对协议
- 领取邮件
- 补给站
- 外部招募
- 退出游戏
- 活动-每日行动补给
